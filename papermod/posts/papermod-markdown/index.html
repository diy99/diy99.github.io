Hugo 與 Markdown#

Markdown 是一種輕量級標記式語言，具易讀易寫特性，並且對於圖片，圖表、數學式都有支援，目前許多網站都廣泛使用Markdown來撰寫說明文件或是用於論壇上發表訊息。學習Markdown可參閱 Markdown Tutorial - Introduction 及 Markdown文件

Hugo 使用 Markdown 編寫文章，更多資訊請參閱 Configure markup | Hugo

PaperMod 與 Markdown#

Hugo 主題作者都會提供 Markdown 範例網頁甚至原始碼，以利安裝者使用參考

PaperMod 與 HTML#

 只使用 Markdown 語言，會覺得無色彩，因此就要加入 HTML，給它好看🌺
 因為我的 PaperMod 設置 config.yml (這也是 Hugo 與 Markdown 的 Configure markup | Hugo) 128至133行是

markup:
  goldmark:
    renderer:
      unsafe: true
  highlight:
    noClasses: false

∴PaperMod Markdown 可以混搭 HTML
 span 是經常使用的，可參閱 HTML span Tag 及 HTML Color Picker ，例如

<span style="font-size:32px;font-weight:bold;color:#0000FF;">傳統</span>_中文_<br>
<span style="font-size:32px;font-weight:bold;color:#FF0000;">简体</span>**中文**
